# Unit Converter

This is a Java Swing based Unit Converter application developed for converting different types of units through a simple user interface.

The project performs fast and accurate conversions for commonly used measurement units.

## Features

- Length Conversion
- Weight Conversion
- Temperature Conversion
- Time Conversion
- Speed Conversion
- Currency Conversion
- User-Friendly Interface

## Technologies Used

- Java
- Swing / AWT
- Eclipse IDE

## Supported Conversions

### Length
- Meter to Kilometer
- Kilometer to Meter
- Centimeter Conversion

### Weight
- Gram to Kilogram
- Kilogram to Gram

### Temperature
- Celsius to Fahrenheit
- Fahrenheit to Celsius

### Time
- Seconds to Minutes
- Minutes to Hours

## Tools Used

- Eclipse IDE
- Java JDK
